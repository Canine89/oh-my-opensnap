# oh-my-opensnap 설치하기 (터미널 필요 없음)

oh-my-opensnap은 Apple Developer ID로 서명되고 공증된 DMG로 배포됩니다.
아래 순서대로 클릭만 하면 설치할 수 있습니다.

> 필요 환경: **macOS 26 (Tahoe) 이상**

---

## 1단계 · 내려받아 설치

1. [**Releases 페이지**](../../releases/latest) 에서 최신 `.dmg` 파일을 내려받습니다.
2. 내려받은 **`.dmg` 파일을 더블클릭**합니다.
3. 창이 열리면 **`oh-my-opensnap` 아이콘을 옆의 `Applications` 폴더로 드래그**합니다.

   ```
   ┌─────────────────────────────┐
   │                             │
   │   🖼  oh-my-opensnap   ─────▶   📁  Applications
   │                             │
   └─────────────────────────────┘
        (왼쪽 아이콘을 오른쪽 폴더로 끌어다 놓기)
   ```
4. 설치 끝. 위쪽 창은 닫고, `.dmg` 는 버려도 됩니다.

---

## 2단계 · 앱 실행

`Applications` 폴더에서 **oh-my-opensnap**을 더블클릭합니다.

앱과 DMG는 Apple Developer ID로 서명되고 공증되어 배포됩니다. 일반적인 설치 과정에서는 “확인되지 않은 개발자” 우회가 필요하지 않습니다.

---

## 3단계 · 화면 녹화 권한 허용

oh-my-opensnap은 화면을 캡처하는 앱이라 권한이 필요합니다. 처음 캡처를 시도하면 권한을 요청합니다.

1. **시스템 설정 → 개인정보 보호 및 보안 → 화면 및 시스템 오디오 녹화** 로 갑니다.
2. 목록에서 **`oh-my-opensnap` 을 켭니다(토글 ON).**
3. 안내가 나오면 앱을 종료했다 다시 엽니다.

> 캡처 단축키 기본값은 **⌘ + ⇧ + 2** 입니다. 메뉴바의 oh-my-opensnap 아이콘에서도 캡처할 수 있어요.

---

## 업데이트는 자동입니다 🔄

새 버전이 나오면 앱이 알아서 알려주고, **`업데이트 설치` 버튼 한 번**이면 끝입니다.
직접 확인하려면 메뉴바 아이콘 → **`업데이트 확인…`** 을 누르세요.

---

## 잘 안 될 때

**"손상되었거나 휴지통으로 옮겨야 한다"고 나와요**
→ 다운로드가 덜 됐거나 깨진 경우입니다. `.dmg` 를 지우고 Releases에서 다시 받아 보세요.

**열리지 않거나 보안 경고가 떠요**
→ 오래된 파일을 받았거나 다운로드가 깨졌을 수 있습니다. 최신 Release의 `.dmg`를 다시 받아 설치해 보세요.

**메뉴 막대(상단 바)에 아이콘이 안 보여요**
→ oh-my-opensnap은 Dock이 아니라 **화면 맨 위 메뉴 막대**에 카메라 아이콘으로 떠 있습니다. 아이콘이 많아 가려졌다면 메뉴 막대를 정리해 보세요.
