# 변경 이력

릴리스마다 `## <버전>` 섹션을 추가하세요. `release.sh`가 해당 섹션을 읽어
Sparkle 업데이트 창과 GitHub 릴리스 노트에 그대로 보여줍니다.

## 1.0.36
- 캡처 영역을 핸들로 조절하거나 이동하는 동안 선택 HUD가 영역을 실시간으로 따라다니도록 개선 (기존에는 조절이 끝난 뒤에만 이동)

## 1.0.35
- 영역을 선택하면 조정 핸들과 함께 선택 HUD(이미지 캡처/영상 촬영/취소)가 바로 표시 — Enter 안내 문구 제거, HUD 버튼이 곧 캡처 확정
- HUD가 떠 있는 동안에도 핸들로 영역을 계속 수정할 수 있고, 버튼을 누르는 시점의 최종 영역으로 캡처/녹화

## 1.0.34
- 캡처 영역을 드래그한 뒤 바로 캡처하지 않고 조정 단계로 진입 — 점선 테두리와 8개 핸들로 영역을 정확히 확인·수정한 뒤 Enter 또는 더블클릭으로 캡처 확정
- 핸들 드래그 중 확대경을 띄워 픽셀 단위로 경계 조절 가능, 영역 내부 드래그로 위치 이동
- 조정 중에는 캡처되지 않는 주변을 더 어둡게 표시해 캡처 범위를 명확하게 구분

## 1.0.33
- 라이브러리 왼쪽 썸네일 목록에서 파일명과 확장자를 함께 표시해 이미지/영상/GIF 항목을 구분하기 쉽게 개선

## 1.0.32
- 영상 촬영 중 선택 영역 바깥을 어둡고 흐린 오버레이로 덮어 현재 녹화 범위를 쉽게 확인할 수 있도록 개선
- 라이브러리 영상 미리보기에서 Space 키로 재생/일시정지를 토글할 수 있도록 지원

## 1.0.31
- 캡처 선택 HUD에서 영상 촬영을 고를 수 있으므로 상태바 메뉴의 별도 `영상 촬영` 항목 제거

## 1.0.30
- 선택 HUD와 촬영 중 HUD의 버튼을 고대비 전용 버튼으로 바꿔 어두운 배경에서도 글자가 확실히 보이도록 개선

## 1.0.29
- 캡처 영역을 선택한 뒤 `이미지 캡처` 또는 `영상 촬영`을 고르는 선택 HUD 추가
- 선택 HUD가 실제 이미지 캡처나 영상 녹화에 섞이지 않도록 선택 후 닫힌 다음 작업을 시작하도록 처리

## 1.0.28
- 영상 편집에서 시작/끝 핸들을 드래그할 때 재생 화면도 해당 시점으로 이동해 선택 장면을 바로 확인할 수 있도록 개선

## 1.0.27
- 라이브러리에서 영상을 볼 때 macOS 기본 플레이어 HUD가 커스텀 편집 HUD 위에 뜨지 않도록 제거

## 1.0.26
- 영상 촬영 HUD를 고정 다크 배경으로 바꿔 글자와 버튼 대비를 개선
- 영상 편집의 MP4 내보내기 문구를 `선택 구간 MP4 저장`으로 바꿔 선택한 초 구간만 새 파일로 저장된다는 의미를 명확히 표시

## 1.0.25
- 영상 촬영 HUD가 다른 창이나 화면을 클릭해도 사라지지 않도록 패널 표시 방식을 보강
- 상태바 메뉴의 `촬영 일시정지` 항목을 실제 영상 촬영 중일 때만 보이도록 개선

## 1.0.24
- 영상 편집의 구간 잘라내기 HUD를 단일 타임라인과 시작/끝 핸들 기반으로 개선
- 라이브러리 영상 편집 창의 기본 크기를 키워 재생과 구간 편집 컨트롤이 눌리지 않도록 조정
- 업데이트 확인 창이 뒤에 숨지 않도록 앱을 활성화하고 Sparkle 업데이트 창을 앞으로 가져오도록 보강

## 1.0.23
- 영상 편집의 잘라내기를 화면 영역 크롭이 아니라 시작/끝 초 구간 잘라내기로 변경
- 현재 재생 위치를 구간 시작/끝으로 지정하는 버튼 추가
- GIF 내보내기도 선택한 초 구간 기준으로 30프레임/45프레임 무한 반복 GIF를 만들도록 변경

## 1.0.22
- 라이브러리에서 GIF 항목을 WebKit 기반 미리보기로 표시해 애니메이션이 재생되도록 수정
- 영상 편집의 잘라내기 흐름을 `영역 선택` → 드래그 → `선택 영역 잘라내기` 순서로 명확히 변경
- 영상 재생 컨트롤을 평소에는 가리지 않도록 잘라내기 오버레이가 영역 선택 모드에서만 마우스 입력을 받게 수정
- 선택 영역이 없을 때 MP4 잘라내기가 전체 영상으로 실행되지 않도록 방지

## 1.0.21
- 영상 촬영 HUD를 마우스로 끌어 원하는 위치로 옮길 수 있게 개선
- 녹화 HUD가 실제 녹화 파일에 잡히지 않도록 캡처 제외 설정 유지
- 녹화 종료 후 라이브러리에서 MP4를 바로 선택해 간단 편집 가능
- 영상 편집에서 지정 영역 MP4 잘라내기와 무한 반복 GIF 내보내기 지원
- GIF 내보내기에 30프레임, 45프레임 옵션 추가
- 모자이크 도구가 더 명확하게 가려지도록 블록 크기와 다운샘플 강도 조정

## 1.0.20
- 영상 촬영에서 창을 클릭하면 정지 캡처처럼 동작하던 문제 수정: 하이라이트된 창 영역도 영상 녹화로 시작

## 1.0.19

- 녹화 HUD와 메뉴바에서 일시정지/재개/중지 제어 지원

## 1.0.18

## 1.0.15
