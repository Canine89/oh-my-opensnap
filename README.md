# oh-my-opensnap — 편집자의 빨간펜 ✍️

화면을 정밀하게 영역 캡처하고, **그 자리에서 빨갛게 표시(번호·화살표·도형)하고 잘라내는** macOS 메뉴 막대 앱.
캡처하면 곧바로 클립보드에 복사되고, 캡처본은 라이브러리에 모여 다시 보고 편집할 수 있습니다.

> macOS 26 (Tahoe) 이상 · 화면 맨 위 메뉴 막대에 상주

---

## 📥 설치

1. [**Releases**](../../releases/latest) 에서 `oh-my-opensnap-x.x.x.dmg` 를 받습니다.
2. `.dmg` 를 열고 **`oh-my-opensnap` 아이콘을 `Applications` 폴더로 드래그**합니다.
3. **처음 한 번만**: 앱을 더블클릭하면 막힐 수 있습니다. 그럴 땐
   **시스템 설정 → 개인정보 보호 및 보안** 으로 가서 아래쪽의 **`그래도 열기`** 를 누르면 실행됩니다.
4. 첫 캡처 때 뜨는 **화면 녹화 권한**을 켭니다
   (시스템 설정 → 개인정보 보호 및 보안 → 화면 및 시스템 오디오 녹화 → `oh-my-opensnap` ON).

> 그림과 함께 한 단계씩 따라 하려면 👉 [INSTALL.md](INSTALL.md)

---

## ✨ 주요 기능

- **영역 캡처**: 전체 화면 디밍 + 크로스헤어 + 라이브 확대경(픽셀 격자·좌표·HEX 색상), 실시간 W×H(px) 표시
- **즉시 클립보드 복사** — 캡처와 동시에 붙여넣기 준비 완료
- **라이브러리 + 편집기** — 캡처본을 모아 보고 빨간펜으로 표시/크롭
- 멀티 디스플레이 / Retina 대응

## 🧭 메뉴 막대 메뉴

메뉴 막대의 카메라 아이콘(📷)을 누르면:

| 메뉴 | 단축키 | 설명 |
|---|---|---|
| **캡처** | `⌘⇧2` | 영역 캡처 시작 (드래그로 범위 지정, `Esc` 취소) |
| **라이브러리…** | `⌘L` | 캡처 보관함 + 편집기 열기 |
| **설정…** | `⌘,` | 캡처 단축키 변경 · 파일 저장 옵션/폴더 · 캡처 사운드 on/off |
| **종료** | `⌘Q` | 앱 종료 |

---

## 🖍 라이브러리 편집기 사용법

`⌘L` 로 라이브러리를 열면 왼쪽에 캡처 목록(썸네일), 오른쪽에 미리보기/편집 화면이 나옵니다.

**상단 도구 막대**

| 도구 | 설명 |
|---|---|
| 🔘 선택 | 편집 안 함(보기 전용) |
| ✂️ 크롭 | 드래그로 영역 지정 → `⏎`(또는 완료)로 잘라내기 |
| ➊ 번호 | 클릭할 때마다 1→9 순서로 동글 번호 찍기. 번호가 커서를 따라다니다 클릭한 자리에 찍힘 |
| ↗️ 화살표 | 드래그로 화살표 |
| ⬛ 사각형 / ⬭ 원 | 드래그로 도형 |
| 🎨 색 / 굵기 | 펜 색상·선 굵기 (기본 빨강) |
| ↩️ 되돌리기 | 마지막 작업 취소 (`⌘Z`, **크롭도 되돌려짐**) |
| 💾 저장 / 📁 Finder / 🗑 삭제 | 다른 곳에 저장 · Finder에서 보기 · 휴지통 |

**알아두면 편한 단축키 / 동작**

- 도형 그릴 때 **`⇧`** 누르고 드래그 → 사각형·원은 1:1, 화살표는 45° 반듯하게
- **`⌘Z`** 되돌리기 · **`⌘C`** 현재 화면 복사
- **`⌘ +` / `⌘ -` / `⌘ 0`** 확대 / 축소 / 창에 맞춤 (`⌘`+스크롤도 가능)
- 썸네일 **우클릭 → Finder에서 보기**
